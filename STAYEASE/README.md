# StayEase - Luxury Hotel Booking Website

StayEase is a responsive hotel booking frontend built using HTML, CSS, and Vanilla JavaScript.

## Features

- Fetch hotels from REST API
- Search hotels by name
- View hotel details
- Responsive UI
- Luxury black & gold theme
- Favorites using Local Storage
- Hotel image gallery
- Mobile friendly

## Technologies

- HTML5
- CSS3
- JavaScript (ES6)
- Fetch API

## API

https://demohotelsapi.pythonanywhere.com/hotels/
